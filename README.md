> [!IMPORTANT]
> ## 🎉 Better Together: JxInsta + instagram4j
> Great news for the developer community! **JxInsta is "coming home."**
> Originally created to pick up where instagram4j left off, JxInsta has now officially merged with the original project. By collaborating directly with the instagram4j maintainers, we are focusing all our energy on building the most robust Instagram private API library for Java.
> ### 🛠 Where to go from here?
> Please redirect all issues, pull requests, and usage to:
> 
> 👉 Official [instagram4j](https://github.com/instagram4j/instagram4j/) Repository
>
